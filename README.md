# Instructions for setup
- Завантажити kaggle API ключ на [kaggle](https://www.kaggle.com/)
- Помістити в папку ./Users/Ваше-ім'я/.kaggle завантажений kaggle.json

### В терміналі:
- pip install kaggle
- kaggle datasets download -d prakharrathi25/google-play-store-reviews
 ---
- Видобути з .zip прям в корінь проєкту -> review.csv в корені проєкту
